Arduino-Teensy-MP3-lib
======================

Audiolibrary plugin, plays up to 320 kbps *.MP3, *.MP4, *.M4A, or *.AAC in software without external hardware - 
with only 48MHz.

Optimized for ARM Thumb2.

planned:
 - *.flac
 - streaming audio
